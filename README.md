# python-skimage

Scikit-image を使用してみる。
（参考: https://www.udemy.com/course/image_processing_python/ ）


## 準備

### pipenv

[pipenv](https://pipenv-ja.readthedocs.io/ja/translate-ja/) を使用しています。

```sh
pip install pipenv
```

を事前におこなっておくこと。


## 実行

```sh
pipenv shell
```

を立ち上げて

```sh
pipenv install
```

をおこないます。

以下のように、フォルダ名を指定して実行します。

```sh
python practice1
```
